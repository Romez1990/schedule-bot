# Schedule bot

Schedule bot was created to give out timetables to students who study in Mephi

## Installation

Use git clone for installing.

```bash
git clone https://github.com/DiZiNnEs/schedule-bot-dizzinnes.git
cd schedule-bot-dizzinnes
pipenv install
```

## Usage

```bash
python3 runner.py
```
